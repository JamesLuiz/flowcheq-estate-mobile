# Flowcheq Agent (mobile)

Native app for **field agents only** — GPS capture, identity verification, and wallet.

- House hunters, landlords, and lawyers → [flowcheq-user-pwa](../flowcheq-user-pwa) or [flowcheq-web](../flowcheq-web)
- Cross-client feature matrix → [docs/FEATURE_PARITY.md](./docs/FEATURE_PARITY.md)

## Setup

```bash
cp .env.example .env
yarn install
yarn start
```

## Env

- `EXPO_PUBLIC_API_URL` — backend API URL
